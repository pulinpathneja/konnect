Web App (Easiest - Works on Any Phone)

  1. Open konnect-app.web.app on your phone browser
  2. Sign in with Google or use Demo mode
  3. Optional: Tap "Add to Home Screen" in browser menu for app-like experience

Android Installation

  1. Download the APK file on your Android phone
  2. Open the downloaded file
  3. If prompted, allow installation from unknown sources in Settings
  4. Tap "Install" and wait for installation to complete
  5. Open the "Konnect" app and sign in with Google or use Demo mode

iOS Simulator Installation

  1. Download and unzip the file on your Mac
  2. Open Xcode and launch an iPhone Simulator
  3. Drag and drop Konnect.app onto the simulator window
  4. Or run: xcrun simctl install booted Konnect.app
  5. Open the Konnect app in the simulator

Testing Checklist

  1. Login: Use "Continue as Demo User" or Google Sign-In
